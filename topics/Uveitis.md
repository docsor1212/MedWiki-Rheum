## 疾病概述

**儿童非感染性葡萄膜炎**

- **定义**: 16岁以下儿童发生的虹膜、睫状体、脉络膜及玻璃体的非感染性炎症
- **流行病学**:
  - 发病率: 4.3-6.8/10万儿童/年
  - 占所有葡萄膜炎的5-10%
  - JIA是最常见的相关全身疾病（占儿童葡萄膜炎的20-40%）
  - 2/3 JIA相关葡萄膜炎为无症状（"安静的白眼"）
- **好发人群**:
  - ANA(+)少关节型JIA: uveitis风险最高（发生率约20-30%）
  - 女性多于男性
  - 发病年龄: 4-8岁高峰

### 病因与分类

**按解剖分型**:
| 类型 | 部位 | 占比 |
|------|------|------|
| 前葡萄膜炎 | 虹膜+睫状体 | 最常见（JIA相关多为前葡萄膜炎） |
| 中间葡萄膜炎 | 睫状体平坦部+玻璃体基底部 | 少见 |
| 后葡萄膜炎 | 脉络膜+视网膜 | 罕见 |
| 全葡萄膜炎 | 全层 | 可见于sJIA、BD、Vogt-小柳原田 |

**按病因分型**:
1. JIA相关（最常见）: ANA(+)少关节型 > RF(-)多关节型 > psoriatic > enthesitis-related
2. 特发性（约30%）
3. 感染性（TB、弓形体、梅毒、病毒 — 必须排除）
4. 全身疾病相关: 川崎病、BD、 sarcoidosis、TINU综合征、Blau综合征

### 病理生理

- 自身免疫性炎症，以T细胞介导为主
- 眼内免疫特权被打破
- TNF-α、IL-6、IL-17是关键促炎因子
- 慢性炎症 → 并发症（白内障、青光眼、带状角膜变性、黄斑水肿）

## 诊断标准

### JIA相关葡萄膜炎筛查（ACR推荐）

**高危人群**: ANA(+)少关节型JIA, 起病年龄<7岁

| 状态 | 筛查频率 |
|------|----------|
| 确诊JIA后 | 立即眼科检查 |
| 高危且前3次正常 | 每3个月1次 |
| 高危且持续正常>4年 | 可延长至每6个月 |
| 高危且有炎症 | 每4-6周直至缓解 |
| 低危（ANA阴性、多关节型） | 每6-12个月 |

### 诊断要素

1. **裂隙灯检查**: 细胞+闪辉（SUN分级标准0-4+）
2. **眼压**: 升高提示炎症或激素反应
3. **眼底检查**: 黄斑水肿、视乳头水肿、视网膜血管炎
4. **OCT**: 黄斑水肿的金标准
5. **实验室检查**:
   - 必需: ANA, ACE, 结核（T-SPOT或PPD）
   - 可选: HLA-B27（enthesitis-related JIA）, HLA-B5/B51（BD）, 弓形体/梅毒/HIV
   - 影像: 胸部CT（排除sarcoidosis/TB）

### SUN分级标准（前房细胞+闪辉）

| 分级 | 细胞数（裂隙灯1mm光束） |
|------|------------------------|
| 0 | <1 |
| 0.5+ | 1-5 |
| 1+ | 6-10 |
| 2+ | 11-20 |
| 3+ | 21-50 |
| 4+ | >50 |

### JIA-Uveitis活动度评分（WGA/WDE）

**前房炎症活动**:
| 分值 | 描述 |
|------|------|
| 0 | 无细胞 |
| 0.5+ | 1-5个细胞 |
| 1+ | 6-10个细胞 |
| 2+ | 11-20个细胞 |
| 3+ | 21-50个细胞 |
| 4+ | >50个细胞 |

**并发症评分**: 各1分（白内障/带状角膜变性/青光眼/黄斑水肿/视盘水肿）

## 治疗策略

### 治疗目标
- 炎症控制至0细胞（完全缓解）
- 防止不可逆并发症（视力损害）
- 保存视功能

### 分层治疗方案

**Step 1: 局部治疗**
- 糖皮质激素滴眼液: 醋酸泼尼松龙1% q1-2h起始，逐渐减量（≤4次/d时维持）
- 散瞳剂: 复方托品酰胺/阿托品（防止虹膜后粘连）
- 适用: 轻度前葡萄膜炎，无后节受累

**Step 2: 局部激素+系统性DMARDs**
- MTX: 10-15mg/m²/wk（皮下或口服）— 首选csDMARD【A级证据】
- 起效时间: 4-12周
- 如果4-6周局部控制不佳即加MTX

**Step 3: 生物制剂（MTX失败或不耐受）**
| 药物 | 剂量 | 证据 |
|------|------|------|
| 阿达木单抗(Adalimumab) | 20mg q2w SC（<30kg）/ 40mg q2w SC（≥30kg） | **A级** (SYCAMORE, ADJUVITE试验) |
| 英夫利昔单抗(Infliximab) | 5-10mg/kg IV 第0/2/6周，后q4-8w | **B级**（抗药抗体风险） |
| 托珠单抗(Tocilizumab) | 8mg/kg IV q4w（<30kg: 12mg/kg） | **C级**（病例系列） |

**阿达木单抗是JIA葡萄膜炎一线生物制剂**（FDA已批准适应症，SYCAMORE RCT阳性）

**Step 4: 系统性糖皮质激素**
- 口服泼尼松: 0.5-1mg/kg/d（短期桥接）
- IVMP冲击: 10-30mg/kg/d × 3天（严重病例）
- ⚠️ 长期使用注意生长发育、白内障、青光眼

**Step 5: 手术**
- 白内障摘除+IOL植入（炎症静止≥3个月）
- 青光眼手术（药物控制不佳时）
- 玻璃体切割术（严重后节受累）

### 并发症处理

| 并发症 | 处理 |
|--------|------|
| 白内障 | 炎症控制≥3个月后手术 |
| 青光眼 | 局部降眼压药 → 手术 |
| 带状角膜变性 | 角膜新生血管抑制（贝伐单抗点眼） |
| 黄斑水肿 | 后节Tenon囊下注射TA或玻璃体注射地塞米松植入剂(Ozurdex) |
| 虹膜后粘连 | 预防为主（散瞳），YAG激光虹膜切除 |

## 随访要点

| 阶段 | 眼科频率 | 监测指标 |
|------|---------|----------|
| 活动期 | 每4-6周 | 裂隙灯细胞+闪辉、眼压、视力、OCT |
| 缓解中 | 每6-8周 | 同上 |
| 缓解后 | 每3个月 | 同上 |
| 稳定>2年 | 每4-6个月 | 同上 |

**预后指标**:
- 炎症持续时间是并发症的最强预测因子
- 诊断延迟>2年 → 白内障风险增加3-5倍
- 20%最终视力损害（如果治疗不及时）

**需紧急就诊**:
- 突发视力下降
- 眼红+痛+畏光加重
- 眼压急骤升高

## 证据等级

- JIA葡萄膜炎筛查指南: **A级** (ACR 2022, AAP)
- MTX作为首选csDMARD: **A级** (多项RCT)
- 阿达木单抗: **A级** (SYCAMORE RCT n=90, NEJM 2020; ADJUVITE RCT)
- 英夫利昔单抗: **B级** (观察性研究)
- SUN分级标准: **A级** (国际共识, 2005)
- 口服激素长期使用: **C级**（避免使用，证据级别低但临床共识强）
