# ITP — 免疫性血小板减少症（儿科）

> **一句话**: 儿童最常见出血性疾病，80%急性自限，20%慢性需免疫治疗，警惕继发性ITP（SLE/AIHA/Evans）
> **定义**: 血小板<100×10⁹/L + 排除其他原因 → 孤立性血小板减少

---

## 📊 流行病学

| 项目 | 数据 |
|------|------|
| 发病率 | 1–6/10万儿童/年 |
| 峰值年龄 | **2–5岁**（急性） |
| 性别比 | 急性M:F≈1:1；慢性女略多 |
| 季节 | 春冬季多发（感染后） |
| 自限率 | 儿童**80%**6月内自发缓解 |

### 急性 vs 慢性

| 特征 | 急性ITP（<12月） | 慢性ITP（≥12月） |
|------|-------------------|-------------------|
| 年龄 | <10岁（尤其2–5岁） | >10岁、青春期 |
| 起病 | 急，感染后1–4周 | 隐匿 |
| 自发缓解 | **80%** | <20% |
| 性别 | M≈F | F>M |
| 抗体 | 一过性 | 持续性 |
| 治疗 | 多数仅需观察 | 需免疫治疗 |

---

## 🔍 诊断

### 诊断标准（排除性诊断）

**必备条件：**
1. 外周血血小板 **<100×10⁹/L**（至少2次）
2. 血涂片：血小板减少，**无异常细胞**，红细胞形态正常
3. 骨髓（非必须）：巨核细胞正常或增多

**必须排除：**
- 假性血小板减少（EDTA抗凝所致 → 用枸橼酸钠重测）
- 遗传性血小板减少（Wiskott-Aldrich、Bernard-Soulier、MYH9相关病）
- 再生障碍性贫血 / MDS / 白血病
- DIC / TTP-HUS
- 药物性血小板减少（肝素、万古霉素、磺胺等）
- 先天性/新生儿同种免疫性血小板减少（NAIT）

### 诊断检查

| 检查 | 目的 | 备注 |
|------|------|------|
| **CBC+涂片** ⭐ | 确认血小板减少+排除异常细胞 | 必须！ |
| 网织血小板 | 鉴别生成减少vs破坏增多 | ↑提示破坏 |
| 抗血小板抗体 | PAIgG（敏感性高，特异性低） | 不作为常规 |
| ANA | 筛查SLE等自身免疫病 | **青春期女必查** |
| dsDNA/抗Sm | 排除SLE | ANA(+)时加查 |
| aPL（抗磷脂抗体） | 筛查APS | 反复血栓+血小板低 |
| 直抗（Coombs） | 排除Evans综合征 | **必查** |
| HBV/HCV/HIV | 治疗前筛查 | 慢性ITP必查 |
| Ig水平 | 排除CVID等免疫缺陷 | 慢性/反复感染 |
| 骨髓 | 排除恶性/再生障碍 | **治疗无效/不典型时** |

### ⚠️ 需骨髓检查的红旗征

| 红旗征 | 可能诊断 |
|--------|----------|
| Hb↓+Ret↓或WBC持续异常 | 白血病/MDS/AA |
| 脾脏明显肿大 | 白血病/淋巴瘤/储存池病 |
| 骨痛 | 白血病/神经母细胞瘤 |
| 异常细胞（涂片） | 白血病 |
| 治疗反应差（多种一线无效） | 非ITP |
| >10岁+全身症状 | SLE/自身免疫 |

## 💊 治疗

### 治疗指征

| 血小板水平 | 出血表现 | 处理 |
|------------|----------|------|
| >30×10⁹/L | 无出血/轻微瘀点 | **观察**（无需治疗） |
| 10–30×10⁹/L | 黏膜出血 | 考虑治疗 |
| <10×10⁹/L | 或湿性出血 | **立即治疗** |
| <20×10⁹/L | 无出血但活动量大的儿童 | 个体化 |

### 出血分级（Buchanan评分）

| 分级 | 表现 | 治疗决策 |
|------|------|----------|
| 0 | 无出血 | 观察 |
| 1 | 轻微瘀点/瘀斑（<5处） | 观察 |
| 2 | 中等瘀斑/少量鼻衄 | 考虑治疗 |
| 3 | **黏膜出血**（鼻衄>30min、牙龈、血尿） | **积极治疗** |
| 4 | **严重出血**（颅内、消化道大出血） | **紧急治疗** |

### 一线治疗

#### 观察等待（初始首选）

**适应症：** 血小板>30×10⁹/L + 无出血或轻微瘀点
- 限制剧烈运动/接触性运动
- 避免使用影响血小板功能的药物（阿司匹林、NSAIDs）
- 随访：每周查血小板直至稳定

#### IVIG

| 项目 | 内容 |
|------|------|
| 剂量 | **0.8–1g/kg/d × 1–2天** 或 **2g/kg分2天** |
| 起效 | 24–72小时 |
| 有效率 | **80%**（暂时性） |
| 维持 | 2–4周 |
| 机制 | Fc受体封闭+抗独特型抗体 |

**适用：** 急性出血、需快速提升血小板、手术前

#### 糖皮质激素

| 方案 | 剂量 | 疗程 |
|------|------|------|
| 泼尼松 | **1–2mg/kg/d**（最大60mg） | 2–4周渐减 |
| 地塞米松 | **0.6mg/kg/d**（最大40mg）× 4天 | 脉冲式 |
| IVMP（严重） | **10–30mg/kg/d** × 3天 | 冲击 |

**适用：** 慢性ITP、IVIG无效或替代
**注意：** 儿童急性ITP应**避免长期激素**（生长影响）

#### 抗D免疫球蛋白

| 项目 | 内容 |
|------|------|
| 剂量 | **50–75μg/kg** 单次 |
| 前提 | **RhD(+)** + 脾脏未切除 + Hb正常 |
| 有效率 | 70–80% |
| 起效 | 24–48小时 |
| ⚠️ 警告 | **一过性溶血**（Hb下降10–20g/L） |

### 二线治疗（慢性/难治性ITP）

| 药物 | 剂量 | 有效率 | 备注 |
|------|------|--------|------|
| **利妥昔单抗(Rituximab)** | 375mg/m² qw × 4周 | 30–60%（6月） | B细胞清除，持续缓解率低 |
| **TPO受体激动剂** | | | |
| — 罗米司亭(Romiplostim) | 1–10μg/kg SC qw | **80%** | 长期维持，不治愈 |
| — 艾曲波帕(Eltrombopag) | 25–75mg/d 口服 | **70–80%** | 肝毒性监测 |
| — 海曲泊帕(Hetrombopag) | 2.5–7.5mg/d | 70%+ | 中国已上市 |
| 环孢素A | 3–5mg/kg/d | 30–50% | 长期使用，肾毒性 |
| MMF | 300–600mg/m² bid | 30–40% | 耐受性好 |
| 硫唑嘌呤 | 1–2mg/kg/d | 20–40% | 起效慢（8–12周） |
| **脾切除** | — | **80%**（即刻） | 最后手段，>5岁，感染风险 |

### 治疗流程

```
确诊ITP
  ↓
血小板>30 + 无出血 → 观察 ← 首选！
  ↓（需要治疗）
一线：IVIG ± 激素 → 有效 → 渐减维持
  ↓（无效/复发）
二线：TPO-RA（长期维持）or 利妥昔单抗
  ↓（仍无效）
三线：免疫抑制剂（CSA/MMF/AZA）
  ↓（全部失败）
脾切除（>5岁，最后手段）
```

### ⚠️ 特殊情况

#### ITP + 妊娠相关（青春期女性）
- 避免使用TPO-RA（安全性未知）
- 分娩时血小板>50×10⁹/L目标
- 硬膜外麻醉需>80×10⁹/L

#### 新生儿ITP
- 母亲ITP → 5–10%新生儿血小板低
- 发生时间：出生后1–3天
- 治疗：IVIG ± 激素
- 颅内出血风险：<1%（但血小板<50时需监测）

## 🔴 Evans综合征

### 定义
**自身免疫性溶血性贫血（AIHA）+ 免疫性血小板减少（ITP）**，同时或相继发生。
可进一步发展为**三系自身免疫（AIHA+ITP+AIN）**。

### 临床特征

| 特征 | 内容 |
|------|------|
| 发病率 | 约占ITP的1–3% |
| 年龄 | 双峰：<5岁和10–15岁 |
| AIHA类型 | **温抗体型（wAIHA）为主** |
| Coombs | **直抗(+)**（IgG/C3） |
| 溶血证据 | Ret↑ + LDH↑ + 间接胆红素↑ + 结合珠蛋白↓ |
| 慢性化率 | **>60%**（远高于单纯ITP） |
| 复发率 | 50–70% |

### Evans vs 单纯ITP

| 特征 | 单纯ITP | Evans综合征 |
|------|---------|-------------|
| 受累系 | 仅血小板 | 血小板+红细胞（±粒细胞） |
| Coombs | (-) | **(+)** |
| 溶血 | (-) | **(+)** |
| 慢性率 | 20% | **>60%** |
| 继发SLE风险 | 低 | **高** |
| 治疗反应 | 好 | **差** |
| 脾切除有效率 | 80% | 40–50% |

### Evans综合征治疗

| 方案 | 内容 |
|------|------|
| 急性 | IVIG + 激素（同ITP，但激素更重要） |
| 维持 | 泼尼松1–2mg/kg→渐减至最低有效量 |
| 激素依赖 | **利妥昔单抗**（首选二线，70%有效） |
| 难治性 | CSA / MMF / 西罗莫司 / 脾切除 |
| 输血 | 温抗体交叉配型困难 → **O型洗涤RBC**紧急输注 |

### Evans → 继发自身免疫病风险评估

| 继发病 | 风险 | 随访 |
|--------|------|------|
| **SLE** | **10–20%** | ANA+dsDNA q6–12月 |
| APS（抗磷脂综合征） | 5–10% | aPL定期 |
| CVID | 5% | Ig水平 |
| ALPS | 罕见 | 双阴性T细胞 |

## 🔴 继发性ITP（SLE相关）

### 流行病学

| 项目 | 数据 |
|------|------|
| SLE患者ITP发生率 | **10–25%** |
| ITP为SLE首发表现 | **3–15%** |
| 儿童>SLE成人 | 更常见 |
| 作为诊断线索 | **青少年慢性ITP建议筛查ANA** |

### SLE-ITP vs 原发ITP

| 特征 | 原发ITP | SLE-ITP |
|------|---------|---------|
| ANA | 通常(-) | **(+)** |
| 抗磷脂抗体 | 通常(-) | **常(+)** |
| 补体 | 正常 | **↓（C3/C4）** |
| 多系统受累 | (-) | **(+)** |
| 肾脏受累 | (-) | **可能(+)** |
| 抗dsDNA | (-) | **(+)** |
| 治疗反应 | 好 | **需SLE整体治疗** |

### SLE-ITP治疗

| 层级 | 治疗 | 备注 |
|------|------|------|
| 基础 | **SLE整体治疗**（HCQ+激素） | HCQ降低复发 |
| 一线 | 激素（同原发ITP） | 控制SLE活动 |
| 二线 | **利妥昔单抗** | 同时改善SLE和ITP |
| 二线 | MMF / AZA | SLE维持治疗兼顾ITP |
| 三线 | TPO-RA | 单独提升血小板 |
| 避免 | 脾切除（SLE患者感染风险极高） | |

### ⚠️ 青少年ITP → SLE转化预警

**高危因素（需长期随访）：**
- ANA(+) → SLE转化率**10–30%**
- 补体低（C3/C4↓）
- 抗磷脂抗体(+)
- 女性 + >10岁
- 多系受累倾向
- **建议随访至少5年**，每6–12月查ANA+补体+aPL

## 🚨 急诊处理

### 颅内出血（<1%，但致命）

| 项目 | 内容 |
|------|------|
| 表现 | 头痛+呕吐+意识改变+局灶神经征 |
| 发生率 | **0.1–0.5%** |
| 死亡率 | 50%+（如未及时处理） |
| 紧急处理 | **IVIG 1g/kg + IVMP 30mg/kg + 输注血小板** |
| 头颅CT | 立即 |
| 神经外科 | 会诊 |

### 严重黏膜出血

| 步骤 | 处理 |
|------|------|
| 1 | IVIG 1g/kg（快速输注） |
| 2 | IVMP 10–30mg/kg |
| 3 | **血小板输注**（虽然被破坏，但大剂量可止血） |
| 4 | 抗纤溶：**氨甲环酸** 10mg/kg IV q8h |
| 5 | 重组因子VIIa（rFVIIa）：难治性出血 |

### 活动限制

| 血小板水平 | 限制 |
|------------|------|
| >50 | 正常活动 |
| 30–50 | 避免接触性运动 |
| 10–30 | 限制剧烈活动 |
| <10 | **床上活动为主**，避免外伤 |

## 📅 随访

| 阶段 | 频率 | 内容 |
|------|------|------|
| 急性期（诊断–4周） | 每周 | CBC+出血评估 |
| 恢复期（1–3月） | 每2–4周 | CBC |
| 慢性期（>12月） | 每1–3月 | CBC+ANA/补体（高危） |
| Evans | 每1–2月 | CBC+Ret+Coombs+LDH |
| SLE-ITP | 按SLE随访 | +血小板监测 |

### 预后

| 类型 | 缓解率 | 备注 |
|------|--------|------|
| 急性ITP | **80%**（6月内） | 多数不需治疗 |
| 持续性ITP（3–12月） | 50%最终缓解 | 部分进入慢性 |
| 慢性ITP | 20–30%自发缓解 | TPO-RA改变预后 |
| Evans综合征 | **<30%**持续缓解 | 需长期免疫抑制 |
| SLE-ITP | 随SLE活动 | 长期预后取决于SLE |

## 证据

- 观察等待安全性: **A级** (Intercontinental Study, Blood 2009)
- IVIG vs 激素: **A级** (SITiP RCT, Lancet 2020)
- 利妥昔单抗: **B级** (多项RCT，长期缓解率低)
- TPO-RA: **A级** (多项RCT，儿童PETIT/PETIT2研究)
- 脾切除: **B级** (>5岁，ITP>1年，最后手段)
- 抗D: **A级** (北美儿童常用，中国少用)
