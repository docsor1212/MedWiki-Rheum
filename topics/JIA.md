# JIA — 幼年特发性关节炎

> **一句话**: 16岁以下、关节炎≥6周、排除其他原因的慢性关节炎综合征，共7个亚型
> **发病率**: 1.6-23/10万/年 | **男女比**: sJIA≈1:1, 其余≈1:3 | **双峰**: 1-3岁, 8-12岁

---

## 🏷️ ILAR分类（2001）— 首先确定亚型

| 亚型 | 关节数 | 系统特征 | ANA | RF | HLA-B27 | 关键鉴别 |
|------|--------|----------|-----|-----|---------|----------|
| **全身型 sJIA** | 任意 | 每日高热≥2周+一过性皮疹 | ± | - | - | 排除感染/肿瘤 |
| **少关节-持续** | ≤4个，前6月 | 无 | +++ | - | - | uveitis风险20-30% |
| **少关节-扩展** | 前6月≤4，后>4 | 无 | ++ | - | - | 可转多关节型 |
| **RF(+)多关节** | ≥5个 | 无 | ± | ++ | - | 似成人RA，侵蚀风险高 |
| **RF(-)多关节** | ≥5个 | 无 | ++ | - | ± | 异质性强 |
| **银屑病相关** | 任意 | 银屑病/指炎/甲脱离+一级亲属银屑病 | ± | - | ± | HLA-B27可+ |
| **附着点炎症相关** | 任意 | 附着点炎+≥2项：骶髂关节炎/HLA-B27+/炎性背痛/前葡萄膜炎 | - | - | +++ | 男>女，可进展AS |
| **未分化** | — | 不符以上或符合≥2个 | — | — | — | 需定期重评 |

> ⚠️ 诊断前提: <16岁 + 关节炎≥6周 + 排除其他原因
> ⚠️ sJIA每日发热特征: 39℃↑ → 每日降至正常（烧热节律）

## 🔍 首诊必做检查

| 必须 | 项目 |
|------|------|
| 血液 | CBC+CRP+ESR+肝肾功能+尿常规 |
| 免疫 | ANA+RF+HLA-B27 |
| 影像 | 受累关节X线（正侧位+对侧）+ 关节超声/MRI |
| 眼科 | **所有JIA确诊即查裂隙灯**（ANA+少关节型 → 每3月） |

| 按亚型追加 | 项目 |
|------------|------|
| sJIA | Ferritin+TG+Fbg（排除MAS）+ 血培养+PCT |
| 附着点炎症相关 | 脊柱MRI+骶髂关节MRI |
| 疑似葡萄膜炎 | OCT+眼压 |

## 💊 治疗阶梯

### 第1步 NSAIDs（所有亚型起始）
| 药物 | 剂量 | 最大量 | 频率 |
|------|------|--------|------|
| 萘普生 | 10-15 mg/kg/d | 1g/d | bid |
| 布洛芬 | 30-40 mg/kg/d | 2.4g/d | tid-qid |
| 塞来昔布 | 3-6 mg/kg/d | 400mg/d | bid |
| 美洛昔康 | 0.125 mg/kg/d | 15mg/d | qd |

> ⚠️ 4-6周无效 → 进入第2步

### 第2步 csDMARD（关节受累进展时）
| 药物 | 剂量 | 途径 | 起效 |
|------|------|------|------|
| **MTX** ⭐首选 | 10-15 mg/m²/周（最大25mg） | **SC优于口服** | 4-12周 |
| 补充叶酸 | 5 mg/周（MTX给药后24h） | 口服 | — |
| 柳氮磺吡啶 | 30-50 mg/kg/d | 口服 | 4-8周 |

### 第3步 生物制剂（MTX失败或不耐受）
| 药物 | 靶点 | 剂量 | 频率 | 首选指征 |
|------|------|------|------|----------|
| **阿达木单抗** | TNF-α | ≤30kg: 20mg; >30kg: 40mg | q2w SC | 多关节型/葡萄膜炎⭐ |
| 依那西普 | TNF-α | 0.8 mg/kg（最大50mg） | biw SC | 多关节型（⚠️不用于葡萄膜炎） |
| 托珠单抗 | IL-6R | ≤30kg: 12mg/kg; >30kg: 8mg/kg | q4w IV | **sJIA首选** / TNF失败 |
| 阿巴西普 | CD28 | 10 mg/kg（<75kg） | q4w IV | 多关节型+TNF失败 |
| 托法替布 | JAK | 5mg bid（≥40kg） | bid | ≥2岁多关节型 |

### sJIA特殊路径（不经过MTX）
```
NSAIDs + 激素(PDN 1-2mg/kg/d)
    ↓ 控制不佳
IL-1抑制剂（阿那白滞素 1-2mg/kg/d SC）⭐ 或 IL-6抑制剂（托珠单抗）
    ↓ 仍活动
TNF抑制剂（效果有限，非首选）
    ↓ 难治
托法替布/芦可替尼
```

> **sJIA首选IL-1/IL-6通路**，TNF抑制剂对sJIA效果有限

### 关节内注射
- 曲安奈德: 1 mg/kg/关节（最大40mg）
- 适用: ≤4个关节受累、快速缓解
- 注意: 间隔≥4周、年≤3次/关节

## ⚡ 警示信号

| 紧急 | 处理 |
|------|------|
| sJIA持续高热+退热药无效 | **查Ferritin/TG/Fbg排除MAS** |
| 突发视力下降/眼红痛 | **紧急眼科**（葡萄膜炎急性发作） |
| 关节红肿热痛明显加重 | **排除感染性关节炎**（穿刺培养） |
| 新发出血点/瘀斑+血象下降 | **查MAS** |

## 📅 随访

| 状态 | 频率 | 核心 |
|------|------|------|
| 活动期 | 每2-4周 | JADAS-27+CBC+CRP+肝功 |
| 稳定中 | 每1-3月 | 同上 |
| 缓解期 | 每3-6月 | JADAS-27+生长评估 |
| **眼科** | ANA(+)+少关节型 | **每3月裂隙灯** |

> **Treat-to-Target**: 3月内至少ACR Pedi 30，6月内达到非活动性疾病
