# JIA — 幼年特发性关节炎

**定义**: <16岁 + 关节炎≥6周 + 排除其他原因 | **发病率**: 1.6-23/10万/年 | **男女**: sJIA≈1:1, 其余≈1:3

**ILAR分类（2001）** — 诊断前提：先排除感染/肿瘤/其他风湿病

| 亚型 | 关节 | 核心特征 | ANA | RF | HLA-B27 | ⚠️注意 |
|:-----|:-----|:---------|:---:|:--:|:-------:|:-------|
| **sJIA** | 任意 | 每日高热≥2w+一过性皮疹（39℃↑烧降节律） | ± | - | - | Ferritin/TG/Fbg排MAS |
| 少关节-持续 | ≤4/前6月 | 无系统特征 | +++ | - | - | **葡萄膜炎20-30%**，每3月裂隙灯 |
| 少关节-扩展 | 先≤4后>4 | 可转多关节型 | ++ | - | - | 同上 |
| RF(+)多关节 | ≥5 | 似成人RA，侵蚀风险高 | ± | ++ | - | 预后较差 |
| RF(-)多关节 | ≥5 | 异质性强 | ++ | - | ± | — |
| 银屑病相关 | 任意 | 银屑病/指炎/甲脱离+一级亲属银屑病 | ± | - | ± | — |
| 附着点炎症相关 | 任意 | 附着点炎+≥2项：骶髂炎/HLA-B27+/炎性背痛/前葡萄膜炎 | - | - | +++ | 男>女，可进展AS |
| 未分化 | — | 不符以上或符合≥2个 | — | — | — | 需定期重评 |

**首诊必做**: CBC+CRP+ESR+肝肾功+尿常规+ANA+RF+HLA-B27+关节X线/MRI+**裂隙灯** | sJIA追加Ferritin+TG+Fbg+血培养+PCT | 附着点炎追加脊柱/骶髂MRI

**治疗阶梯**: NSAIDs 4-6w → csDMARD → 生物制剂

| 阶段 | 药物 | 剂量 | 途径 | 频率 | 备注 |
|:-----|:-----|:-----|:-----|:-----|:-----|
| **1. NSAIDs** | 萘普生 | 10-15mg/kg/d | po | bid | max 1g |
| | 布洛芬 | 30-40mg/kg/d | po | tid-qid | max 2.4g |
| | 塞来昔布 | 3-6mg/kg/d | po | bid | max 400mg |
| **2. csDMARD** | **MTX** ⭐ | 10-15mg/m² | **SC首选** | qw | max 25mg，+叶酸1mg/d |
| | 柳氮磺吡啶 | 30-50mg/kg/d | po | bid | 4-8w起效 |
| **3. 生物制剂** | 阿达木单抗 | ≤30kg:20mg >30kg:40mg | SC | q2w | 多关节+葡萄膜炎⭐首选 |
| | 依那西普 | 0.8mg/kg max50mg | SC | biw | ⚠️不用于葡萄膜炎 |
| | 托珠单抗 | ≤30kg:12mg/kg >30kg:8mg/kg | IV | q4w | **sJIA首选** |
| | 阿巴西普 | 10mg/kg <75kg | IV | q4w | TNF失败后 |
| | 托法替布 | 5mg bid ≥40kg | po | bid | ≥2岁 |

**sJIA特殊路径**（不经MTX）: NSAIDs+PDN 1-2mg/kg/d → 控制不佳 → **IL-1**(阿那白滞素1-2mg/kg/d SC) 或 **IL-6**(托珠单抗) → TNF(效果有限) → JAK抑制剂

**关节内注射**: 曲安奈德 1mg/kg/关节 max40mg，间隔≥4w，年≤3次/关节

**警示信号**: 🔴sJIA持续高热退热药无效→查MAS | 🔴突发视力↓/眼红痛→紧急眼科 | 🔴关节红肿热痛加重→排除感染性关节炎（穿刺培养） | 🔴出血点/瘀斑+血象↓→查MAS

**随访**: 活动期q2-4w(JADAS-27+CBC+CRP+肝功) → 稳定q1-3m → 缓解q3-6m | **ANA(+)+少关节型每3月裂隙灯** | Treat-to-Target: 3m内ACR Pedi30，6m内非活动性疾病

**证据**: ACR 2021 JIA治疗更新(Onel KB, Arthritis Rheumatol 2021) | sJIA IL-1/IL-6(De Benedetti F, NEJM 2012; Ruperto N, Lancet 2017) | Treat-to-Target(Anton J, Ann Rheum Dis 2018) | 葡萄膜炎阿达木单抗SYCAMORE RCT(NEJM 2020)
