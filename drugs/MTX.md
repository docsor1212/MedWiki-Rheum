# MTX — 甲氨蝶呤

> **一句话**: 儿童风湿一线csDMARD，JIA/JDM/SLE首选，SC优于口服
> **起效**: 4-12周 | **必须联用叶酸** | **致畸** — 停药≥3月方可妊娠

| 方案 | 剂量 | 说明 |
|------|------|------|

## 🚫 禁忌症

| 绝对禁忌 | 相对禁忌 |
|----------|----------|
| 严重肝肾功能不全 | 酒精性肝病 |
| 妊娠/哺乳/备孕 | 肥胖/糖尿病/NAFLD |
| 已知过敏 | 既往肝毒性 |
| 严重免疫缺陷 | 既往肺毒性 |
| 活动性严重感染 | 消化性溃疡 |

---

## ⚠️ 不良反应

### 常见（可耐受）
| 反应 | 发生率 | 处理 |
|------|--------|------|
| 恶心/呕吐 | 10-30% | 改SC，给药前吃少量食物 |
| 口腔溃疡 | 5-15% | 补叶酸 |
| 转氨酶↑（1-3×ULN） | 10-20% | 监测，常自行恢复 |
| 疲劳 | 10-20% | 调整给药时间至睡前 |

### 严重（需停药）
| 反应 | 阈值 | 处理 |
|------|------|------|
| 骨髓抑制 | WBC<3.0 或 PLT<100 | 立即停药，支持治疗 |
| 肝纤维化 | 持续转氨酶升高>6月 | 停药，肝纤维化评估 |
| MTX性肺炎 | 咳嗽+呼吸困难+发热 | 立即停药，排除感染 |
| PCP肺炎 | 免疫抑制患者 | 预防：TMP-SMX qow |
| 恶性肿瘤 | 极罕见，长期用药 | 监测 |

## 🔬 监测时间表

| 时间 | 检查 |
|------|------|
| **基线** | CBC+肝肾功能+HBV/HCV+胸片+PPD/T-SPOT |
| 每月×3月 | CBC+肝肾功能 |
| 稳定后 | 每2-3月 CBC+肝肾功能 |
| 长期(>5年) | 每年肝纤维化评估 |
| 使用生物制剂期间 | 结核筛查（T-SPOT）定期复查 |

### 停药阈值
| 指标 | 警告线 | 停药线 |
|------|--------|--------|
| WBC | <4.0×10⁹/L | <3.0×10⁹/L |
| PLT | <150×10⁹/L | <100×10⁹/L |
| ALT/AST | >3×ULN | >5×ULN持续2次 |
| 肌酐 | ↑ | CrCl<30ml/min |

## 💊 相互作用

| 药物 | 影响 | 建议 |
|------|------|------|
| **TMP-SMX** ⚠️ | 竞争叶酸代谢→骨髓抑制↑↑ | **禁止联用** |
| NSAIDs | ↑MTX血药浓度 | 监测肾功能，短期可用 |
| PPI(奥美拉唑) | ↓MTX清除 | 换H2RA(法莫替丁) |
| 青霉素类 | ↑MTX毒性 | 监测 |
| 叶酸>5mg/d | 可能↓MTX疗效 | 避免过量补充 |
