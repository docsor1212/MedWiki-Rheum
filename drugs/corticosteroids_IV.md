## 用法用量

### 1. IVMP冲击治疗（Pulse Therapy）

- **剂量**: 10-30 mg/kg/d
- **最大剂量**: 1 g/d
- **溶媒**: 5%葡萄糖或0.9%氯化钠，浓度≤5mg/ml
- **输注时间**: ≥1-2小时（30mg/kg方案建议≥2小时）
- **疗程**: 连续3天，必要时可延长至5天
- **心电监测**: 大剂量时建议（低钾/心律失常风险）

### 2. IVMP标准剂量

- **剂量**: 1-2 mg/kg/d（等效泼尼松）
- **频率**: qd或分2次
- **疗程**: 依疾病，通常3-7天后转为口服

### 3. 激素等效换算

| 药物 | 等效剂量(mg) | 抗炎效力 | 盐皮质激素 |
|------|-------------|----------|------------|
| 氢化可的松 | 20 | 1 | 强 |
| 泼尼松 | 5 | 4 | 弱 |
| 甲泼尼龙 | 4 | 5 | 极弱 |
| 地塞米松 | 0.75 | 25-30 | 极弱 |

### 4. 各疾病推荐方案

#### JDM急性期

```
IVMP 30mg/kg/d × 3天（最大1g/d）
→ 泼尼松口服 1-2mg/kg/d（逐步过渡）
→ 维持6-12周后开始缓慢减量
→ 总疗程≥1-2年
```

#### JSLE狼疮性肾炎（IV型）

```
IVMP 500-1000mg/d × 3天（或30mg/kg/d × 3天）
→ 泼尼松口服 1mg/kg/d（最大60mg/d）
→ 4-8周后开始减量（每2-4周减10%）
→ 维持剂量: ≤0.1mg/kg/d 或隔日
→ 联合MMF或IVCY
```

#### sJIA合并MAS

```
IVMP 30mg/kg/d × 3天（最大1g/d）
→ 泼尼松 2mg/kg/d（可分次）
→ 有效后2-4周开始减量
→ 联合CSA/他克莫司 + Anakinra
```

#### 川崎病（KD）

```
IVMP 2mg/kg/d × 3-5天（KD耐药补救方案）
或 IVMP 30mg/kg/d × 1-3天（冲击方案，KSS首选）
→ 联合IVIG 2g/kg
```

#### 川崎休克综合征（KSS）

```
IVMP 30mg/kg/d × 3天（最大1g/d）
→ 联合IVIG 2g/kg
→ 必要时加用CsA 3-5mg/kg/d
→ 血管活性药物支持
```

## 激素减量原则

### 通用减量策略

1. **急性期控制后**: 每2-4周减量10%
2. **减至0.5mg/kg/d后**: 减速至每月减10%
3. **减至隔日方案**: 维持数月后再减
4. **最小维持剂量**: 泼尼松 ≤0.1mg/kg/d 或 ≤5-7.5mg/d
5. **总疗程**: 依疾病，JDM≥1-2年，SLE≥2-3年

### 减量注意事项

- 减量过快 → 疾病反弹（最常见）
- 关注晨间僵硬、炎症指标、疲乏等反弹信号
- 减量至0.25mg/kg/d时最易反弹，需格外谨慎
- 联合免疫抑制剂可降低反弹风险

## 不良反应（分级）

### 急性不良反应（冲击治疗相关）

- **低钾血症**: 心律失常风险 → 补钾+心电监测
- **高血压**: 一过性
- **高血糖**: 短暂性，停药后恢复
- **面部潮红**: 输注过快相关
- **行为改变**: 欣快、烦躁、失眠
- **胃肠道**: 恶心、消化不良
- **过敏反应**: 极罕见（含亚硫酸盐制剂）

### 短期不良反应（口服>2周）

- **满月脸、水牛背**（Cushing征象）
- **体重增加、食欲亢进**
- **痤疮、多毛**
- **血糖升高** → 监测空腹血糖
- **高血压** → 监测血压
- **肌无力**（近端肌病）
- **情绪障碍**: 烦躁、抑郁
- **骨质疏松**: 早期即开始，累积剂量相关

### 长期不良反应（>3月）

- **生长障碍**: 最关注！线性生长↓
- **骨质疏松/股骨头坏死**: 需补充钙+维生素D
- **白内障/青光眼**: 定期眼科检查
- **肾上腺抑制**: 停药时需逐渐减量
- **机会性感染**: PCP → 建议TMP-SMX预防
- **皮肤萎缩、紫纹**
- **脂肪重新分布**
- **胰岛素抵抗/继发性糖尿病**

### 感染预防

- **PCP预防**: 泼尼松≥0.5mg/kg/d ≥4周 → TMP-SMX预防
- **结核**: 用药前筛查TST/IGRA，阳性需预防性抗结核
- **疫苗接种**: 避免减活疫苗（激素>2周期间）

## 监测指标（时间表）

| 时间 | 检查项目 |
|------|----------|
| 冲击前 | 血压、血钾、血糖、心电图（大剂量）、体重 |
| 冲击中 q4h | 血压、心率、血糖 |
| 口服期每月 | 血压、体重、血糖（或HbA1c q3月）、血钾 |
| 口服期每3月 | 生长曲线、骨龄（>6月）、眼科检查 |
| 长期 | 骨密度（>1年或累积剂量高者）、腰椎评估 |

## 禁忌症

- **系统性真菌感染**
- **活动性结核（未治疗）**
- 对激素过敏（极罕见）
- **相对禁忌**: 消化性溃疡（活动性）、未控制的高血压、未控制的糖尿病、精神病史

## 药物相互作用

- **CYP3A4诱导剂**: 苯巴比妥、利福平、苯妥英 → 激素代谢↑，需增量
- **CYP3A4抑制剂**: 酮康唑、克拉霉素 → 激素代谢↓，毒性↑
- **利尿剂（排钾）**: 加重低钾血症
- **NSAIDs**: 消化道出血风险叠加
- **疫苗**: 减活疫苗在激素治疗期间禁用
- **华法林**: 激素影响凝血因子，INR波动

## 特殊人群

### 肝功能不全

- 地塞米松主要经肝脏代谢 → 肝功能不全时半衰期延长
- 甲泼尼龙: 轻中度不全无需调整

### 肾功能不全

- 无需调整剂量
- 注意水钠潴留、高血压加重

### 妊娠/哺乳

- 泼尼松/甲泼尼龙可跨胎盘（约10%达到胎儿）
- 妊娠期重症风湿病可用（获益>风险）
- 产程中需应激剂量覆盖
- **哺乳**: 泼尼松少量进入乳汁（<0.1%母体剂量），通常可哺乳
