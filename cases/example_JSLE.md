# 示例病例：儿童系统性红斑狼疮 (JSLE)

> 本病例为教学示例，展示初诊→检查→诊断→治疗→随访完整流程
> 更新: 2026-04-08

## 基本信息

- 患儿: 王XX，女，12岁3月
- 住院号: SLE-2026-001
- 就诊日期: 2026-01-20

## 主诉

反复面部红斑、口腔溃疡3月，发热伴双膝关节疼痛2周。

## 现病史

患儿3月前无明显诱因出现双颊部蝶形红斑，日晒后加重，伴反复口腔溃疡（上颚、颊黏膜），每月发作2-3次，自行可愈合。2月前出现脱发，梳头时明显增多。家人未重视。

2周前出现低热，体温37.5-38.2℃，伴双膝关节疼痛，无肿胀。同时发现面部及双下肢水肿，尿量减少（约500mL/d）。无光过敏、雷诺现象、关节畸形。

**既往史**: 体健，预防接种按期完成。
**家族史**: 母亲有"干燥综合征"病史。
**过敏史**: 否认药物过敏。

## 诊断

1. **儿童系统性红斑狼疮 (JSLE)** — 2019 EULAR/ACR分类标准 43分
2. **狼疮性肾炎 IV型**（ISN/RPS 2003，弥漫性，AI 8/CI 1）
3. **低补体血症**
4. **继发性抗磷脂抗体综合征**（暂无血栓事件，需预防）

## 病例要点总结

1. JSLE以皮肤（蝶形红斑）、黏膜（口腔溃疡）、肾脏受累最常见，青少年女性好发
2. 2019 EULAR/ACR标准采用积分制（≥10分），比传统标准更敏感
3. SLEDAI-2K用于疾病活动度监测，指导治疗调整
4. LN-IV型诱导方案: IVMP冲击 → 口服激素 + MMF，与成人方案一致
5. MMF是儿童LN一线免疫抑制剂（优于环磷酰胺，生殖毒性更低）
6. 羟氯喹（HCQ）应长期使用，有减少复发、降低血栓风险等多重获益
7. 抗磷脂抗体阳性 → 血栓预防（阿司匹林/低分子肝素）
8. 定期监测: 补体/dsDNA是活动度最佳血清学标志物
