# 示例病例：少关节型JIA (oJIA)

> 本病例为教学示例，展示初诊→检查→诊断→治疗→随访完整流程
> 更新: 2026-04-08

## 基本信息

- 患儿: 李XX，女，3岁2月
- 住院号: JIA-2026-002
- 就诊日期: 2026-02-10

## 主诉

左膝关节肿痛2月，发现跛行1月。

## 现病史

患儿2月前家长发现左膝肿胀，活动后哭闹，起初间歇性，近1月持续肿胀，走路跛行。无发热、皮疹、晨僵（家长未关注）。

否认光过敏、口腔溃疡、腹痛、血尿。

**既往史**: 体健，按期接种疫苗。
**家族史**: 母亲有桥本甲状腺炎。
**过敏史**: 否认。

## 体格检查

- T 36.5℃，HR 100次/分，生命体征平稳
- 体重 14kg（P50）
- **左膝关节**: 明显肿胀（髌上囊饱满），皮温稍高，屈伸轻度疼痛，屈曲受限（从150°→130°）
- **右膝关节**: 无异常
- 其余关节检查正常
- 无皮疹，无淋巴结肿大，心肺腹正常
- **肿胀关节数: 1 / 活动受限关节数: 1 / 压痛关节数: 1**

### 影像学

- 左膝超声: 关节腔积液（少量），滑膜增厚
- 左膝X线: 软组织肿胀，无骨质破坏

### 眼科检查

- 裂隙灯: 双眼前葡萄膜炎（左眼1+细胞，右眼正常）→ **无症状uveitis**

## 初步诊断

### ILAR分类标准对照（少关节型）

| 标准条目 | 符合 |
|----------|------|
| 关节炎1-4个关节 | ✅ 1个（左膝） |
| 持续≥6周 | ✅ 约8周 |
| 排除RF阳性 | ✅ RF阴性 |
| 排除全身型 | ✅ 无发热/皮疹/浆膜炎 |
| 排除银屑病 | ✅ |
| 排除附着点炎 | ✅ |
| 排除其他 | ✅ |

**诊断: 少关节型JIA（ANA阳性，持续性）**

**并发症**: 双眼前葡萄膜炎（左眼活动性）

### uveitis风险

- oJIA + ANA(+) + 年龄<4岁 → **高危**，需每3月眼科随访

## 治疗方案

| 药物 | 剂量 | 途径 | 频率 |
|------|------|------|------|
| NSAIDs（布洛芬） | 10mg/kg/d | PO | tid |
| MTX | 15mg/m²/wk（10mg/wk） | PO→SC | 每周1次 |
| 叶酸 | 5mg/周 | PO | MTX给药24h后 |
| 眼局部激素（醋酸泼尼松龙） | 滴眼 | qid | 眼科处方 |

- 关节腔注射（左膝）: 曲安奈德 1mg/kg + 口服MTX同时启动
- 或选择密切观察MTX疗效

## 随访经过

### 第4周（2026-03-10）

- 左膝肿胀明显改善
- ESR 15 mm/h，CRP 4 mg/L
- 眼科: 左眼uveitis 1+→trace，右眼正常
- MTX耐受好，轻度恶心（已加叶酸）
- NSAIDs可逐步减量

### 第12周（2026-05-05）

- 左膝无肿胀，活动正常
- 活动关节数: 0
- ESR 8 mm/h，CRP <4 mg/L
- **JADAS-27: 0.5（达到inactive disease）**
- 眼科: 双眼正常，无uveitis复发
- NSAIDs停用
- MTX继续，泼尼松龙眼药水减量

### 第24周（2026-08-05）

- 维持缓解状态
- 眼科: 正常
- MTX 10mg/wk 继续维持
- **JADAS-27: 0**

## 病例要点总结

1. oJIA以单/少关节肿痛为表现，系统症状少，易延误诊断
2. ANA阳性、年龄小 → uveitis高危，**必须**定期眼科筛查
3. uveitis可无症状但导致严重视力损害
4. MTX是oJIA首选csDMARD，也助于控制uveitis
5. 关节腔注射可快速缓解单关节炎
6. 目标: 达到并维持Wallace inactive disease
