🔴 12月龄男婴以肾病综合征起病的婴儿SLE

全身水肿2月 → IV型狼疮肾炎 + 大面积肺栓塞 — 1岁以内发病的极端罕见SLE
更新: 2026-04-10
⚠️ 免责声明　本病例基于已发表的医学文献整理，已做脱敏处理。仅供医学专业人员学习参考，不构成任何诊疗建议，不可替代临床医师的专业判断。
👤 基本信息
患者12个月，男性
就诊机构安卡拉大学医学院，儿科肾脏科（土耳其安卡拉）
主诉全身水肿（Generalized Edema）2个月
标签婴儿SLE IV型狼疮肾炎 大面积肺栓塞 完全缓解
本病例独特性：① 12个月男婴（cSLE的极端低龄端，婴儿SLE极为罕见）；② 以肾病综合征为首发表现（非典型）；③ ANA阴性（SLE伴ANA阴性极少见，增加了诊断难度）；④ 合并大面积肺栓塞（Massive Pulmonary Thromboembolism）；⑤ 肾活检为IV型狼疮肾炎（弥漫增殖性，最严重类型之一）。
🔴 临床表现
起病 — 全身水肿2个月
12个月男婴，出现全身水肿（Generalized Edema），逐渐加重，持续2个月。
家长和基层医生首先考虑营养不良/低蛋白血症。
入院评估 — 多系统异常
肾脏：
• 肾病范围蛋白尿（Nephrotic-range Proteinuria）
• 低白蛋白血症
→ 符合肾病综合征（Nephrotic Syndrome，NS）

血液系统：
• 溶血性贫血（Hemolytic Anemia）
• 血小板减少（Thrombocytopenia）
• 直接和间接Coombs试验均阳性

免疫学：
• ANA（抗核抗体）— 阴性⚠️（SLE中极不寻常）
• C3 轻度降低
• C1q 降低（补体经典途径早期成分消耗）
• C4 正常
• 抗磷脂抗体（aPL）阴性
• 抗心磷脂抗体（aCL）阴性

ANA阴性的SLE — 诊断极具挑战性
住院期间 — 大面积肺栓塞
住院期间突发：
• 呼吸困难（Dyspnea）
• 缺氧（Hypoxia）
• 影像学（CT肺动脉造影）：大面积肺血栓栓塞（Massive Pulmonary Thromboembolism，PE）

危及生命的并发症 — 肺栓塞
🔬 肾活检 — IV型狼疮肾炎
📋 ISN/RPS 2003/2018 狼疮肾炎分类（简表）
分型病理特征临床意义
I型微小系膜病变光镜正常，免疫荧光可见沉积
II型系膜增殖性系膜细胞增生+免疫沉积
III型局灶性（<50%肾小球）局灶增殖±新月体
IV型 ⬅ 本例弥漫性（≥50%肾小球）最严重常见型：弥漫增殖+内皮下沉积±新月体
V型膜性上皮下沉积，肾病综合征
VI型晚期硬化（≥90%）终末期肾病
本例肾活检结果：
• IV型（弥漫增殖性）狼疮肾炎 — ≥50%肾小球受累
• 免疫荧光：免疫复合物沉积（IgG、IgM、C3、C1q — "满堂亮（Full House）"模式）
→ 这是SLE肾脏受累的最严重且最常见类型，需要积极免疫抑制治疗

为什么这个活检是确诊的关键？
本例ANA阴性，C4正常，aPL阴性 — 仅凭血清学无法诊断SLE。
但肾活检显示典型的"满堂亮"免疫复合物沉积 + IV型弥漫增殖 → 肾脏病理弥补了血清学的不足，成为确诊的关键依据。
✅ 诊断
1. 婴儿SLE（Infantile SLE，iSLE）
依据：肾病综合征+溶血性贫血+血小板减少+C3/C1q降低+Coombs阳性+IV型LN（肾活检）

2. IV型狼疮肾炎（Class IV LN）
依据：肾活检 — 弥漫增殖性肾小球肾炎+"满堂亮"免疫荧光

3. 肾病综合征（Nephrotic Syndrome）
依据：肾病范围蛋白尿+低白蛋白血症+全身水肿

4. 大面积肺栓塞（Massive PE）
依据：CT肺动脉造影确诊
⚠️ aPL阴性但出现PE → 探讨PE的其他可能机制（见下方机制解析）
💊 多阶段治疗方案
阶段1：诱导缓解
糖皮质激素（Steroid）
→ 控制免疫炎症，减少免疫复合物形成

环磷酰胺（CTX，Cyclophosphamide） 静脉冲击
→ IV型LN的诱导缓解金标准（与大年龄儿童方案一致）

抗凝治疗（Anticoagulation）
→ 治疗大面积肺栓塞
→ 即使aPL阴性，肾病综合征本身是高凝状态 → 抗凝指征明确
阶段2：维持缓解
吗替麦考酚酯（MMF，Mycophenolate Mofetil）
→ CTX诱导后转换为MMF维持
→ IV型LN维持缓解的首选药物之一
→ 适用于长期免疫抑制，安全性优于CTX
转归 — 完全缓解
完全缓解（Complete Remission）：
• 蛋白尿消失
• 肾功能正常
• 溶血性贫血纠正
• 血小板恢复正常
• 肺栓塞溶解
12个月男婴 + IV型LN + 大面积PE → 完全缓解
🔬 婴儿SLE + 肺栓塞的机制（深度解析）
1. 婴儿SLE（iSLE）的特殊性
① iSLE = 发病年龄<1岁的SLE，极为罕见（文献仅零星报道）
② iSLE比大年龄cSLE更严重：更高频率的关键器官受累（肾脏、血液、中枢）
③ iSLE曾被认为预后很差，但本例证明积极免疫抑制可获完全缓解
④ iSLE需要与先天性自身免疫病（如Aicardi-Goutières综合征、I型干扰素病）鉴别

2. 为什么ANA阴性？— 补体消耗假说
⑤ 本例ANA阴性但C1q显著降低 → 可能机制：
⑥ C1q被大量免疫复合物消耗 → 补体经典途径持续激活
⑦ ANA可能与免疫复合物结合形成循环免疫复合物 → 游离ANA减少 → 检测阴性
⑧ 另一可能：婴儿期免疫系统发育不成熟 → 自身抗体谱不典型
⑨ 教训：ANA阴性不能排除SLE，尤其在补体降低、多系统受累时

3. C1q降低的特殊意义
⑩ C1q是补体经典途径的起始分子，与免疫复合物的IgG/IgM结合启动级联
⑪ C1q降低 → 提示大量免疫复合物持续消耗补体 = 高度免疫活动
⑫ C1q降低还见于C1q肾病和低补体血症性荨麻疹性血管炎综合征（HUVS）
⑬ 本例C1q低+C3低但C4正常 → 可能提示补体消耗主要发生在经典途径早期阶段

4. aPL阴性为什么出现肺栓塞？— 肾病综合征的高凝状态
高凝因素机制
抗凝血酶（AT）丢失肾病范围蛋白尿 → AT从尿中丢失 → 抗凝能力↓
纤维蛋白原↑肝脏代偿性合成增加 → 血液粘稠度↑
血小板活化高胆固醇+免疫复合物 → 血小板聚集↑
纤溶抑制纤溶酶原激活物抑制剂（PAI-1）↑ → 纤溶↓
免疫复合物SLE免疫复合物损伤内皮 → 组织因子暴露 → 凝血激活
补体活化C5b-9膜攻击复合物 → 内皮损伤 → 促凝表面
🔥 核心教训：肾病综合征本身就是独立的高凝状态，即使没有APS/aPL，也足以导致大面积肺栓塞。儿童肾病综合征合并PE的死亡率高达25-30%。任何肾病综合征患儿出现呼吸困难/胸痛/低氧，必须立即行CT肺动脉造影。

5. "满堂亮"免疫荧光 — SLE肾脏的标志
⑭ 本例肾活检免疫荧光显示IgG、IgM、C3、C1q全部阳性 — 即"满堂亮（Full House）"
⑮ "满堂亮"是SLE肾脏受累的特征性表现（其他肾小球疾病通常只有1-2种）
⑯ 但"满堂亮"并非SLE特异 — 也可见于：
　　• 感染后肾小球肾炎（如乙肝相关）
　　• C1q肾病
　　• 纤维样肾小球肾炎
⑰ 需结合临床+血清学综合判断

6. 婴儿狼疮肾炎的治疗考量
⑱ IV型LN需要积极诱导缓解：首选糖皮质激素+CTX或MMF
⑲ 本例使用CTX诱导 → MMF维持 → 与大年龄cSLE方案一致
⑳ 但12个月婴儿使用CTX的特殊考量：
　　• 性腺毒性（尤其男性患儿，CTX对生精细胞不可逆损伤）
　　• 感染风险（婴儿免疫系统发育中）
　　• 骨髓抑制耐受性更低
㉑ MMF维持是更安全的选择 — 本例成功转换为MMF长期维持
📚 临床要点总结
诊断层面：
① 婴儿肾病综合征 + 多系统受累必须考虑iSLE — 不能因年龄小而排除
② ANA阴性不能排除SLE — 尤其在补体降低时，行肾活检至关重要
③ "满堂亮"免疫荧光高度提示SLE肾脏受累
④ C1q降低是补体经典途径激活的重要线索

治疗层面：
⑤ iSLE的IV型LN治疗方案与大年龄cSLE一致（激素+CTX→MMF）
⑥ 婴儿使用CTX需特别注意性腺毒性和感染风险
⑦ 积极免疫抑制可获完全缓解 — iSLE并非"不治之症"
⑧ 肾病综合征合并PE → 免疫抑制 + 抗凝双管齐下

警示层面：
⑨ 肾病综合征 = 高凝状态，无需aPL阳性也可发生血栓
⑩ 婴儿出现呼吸困难/低氧 → 必须排除PE
⑪ 儿童PE可无明显下肢DVT（深静脉血栓）— "沉默型"PE
