📋 病例5：难治性sJIA

12年病程 — 依那西普失败 → Anakinra有效 → 停药复发 → 托珠单抗+感染触发MAS → 多线转换
更新: 2026-04-10
⚠️ 免责声明　本病例基于已发表的医学文献整理，已做脱敏处理。仅供医学专业人员学习参考，不构成任何诊疗建议，不可替代临床医师的专业判断。具体诊疗方案应由执业医师根据患者实际情况制定。如有疑问请咨询专业医疗机构。
👤 基本信息
患者男性，2岁起病
随访跨度2岁 → 14岁+（12年+）
起病表现sJIA典型表现 + MAS（MAS为首发并发症）
标签难治性 多线转换 反复MAS
本病例特点：完整展示sJIA从幼儿到青少年的全病程管理，包括多线生物制剂转换、停药复发、感染触发MAS、长期糖皮质激素依赖的挑战。
📅 完整治疗时间线（12年）
2岁 — sJIA起病 + MAS
根据ILAR标准诊断为sJIA。
起病时即以MAS为首发表现（典型临床表现+实验室+组织学确诊）。
首次MAS
4岁 — 依那西普（etanercept，TNF-α抑制剂） 失败
启动依那西普治疗。1年后因疗效不佳停用。
TNF-α抑制剂在sJIA中的疗效不如在其他JIA亚型中
8岁 — Anakinra（阿那白滞素，IL-1受体拮抗剂） 有效
启动Anakinra，临床反应良好。持续治疗6年。
这是目前维持时间最长的治疗阶段。
14岁 — Anakinra停药（临床缓解+局部不良反应）
因达到临床缓解且出现注射部位疼痛（局部不良反应）而停药。
停药6个月后 — 疾病复发
出现新的关节炎发作 + 系统性活动（发热等）。
选择托珠单抗（tocilizumab，抗IL-6受体单抗）单药治疗（未联合糖皮质激素）。
托珠单抗第2次输注后 — 上呼吸道感染 → MAS
第2次输注后不久出现上呼吸道感染，口服阿莫西林+非甾体抗炎药。
数日后因MAS入院。
骨髓穿刺：噬血现象阳性
EBV（Epstein-Barr病毒）近期感染
第2次MAS
MAS抢救
方案：环孢素A + 静脉甲泼尼龙冲击
结果：症状完全缓解。
📊 治疗药物评估总结
药物使用时间疗效停药原因教训
依那西普4-5岁（约1年）无效疗效不佳TNF-α抑制剂在sJIA中效果有限
Anakinra8-14岁（6年）有效临床缓解+注射部位疼痛IL-1通路阻断对sJIA有效；停药后可能复发
托珠单抗14岁+MAS治疗中发生MASIL-6阻断不能预防感染触发的MAS
环孢素A+激素MAS发作时有效MAS控制后停用标准MAS挽救方案
📚 病例要点总结
1. sJIA的长期管理挑战
① sJIA可以从幼儿期延续至青少年甚至成年
② 单一药物难以永久控制，需要多线转换
③ 停药后复发风险高（本例停Anakinra 6个月后复发）

2. 生物制剂的选择与转换
④ TNF-α抑制剂（依那西普）在sJIA中疗效有限 — 不是首选
⑤ IL-1抑制剂（Anakinra/卡那单抗）和IL-6抑制剂（托珠单抗）是sJIA一线生物制剂
⑥ 不同靶点的生物制剂转换是合理的临床策略

3. MAS复发风险
⑦ 有MAS病史的患者是MAS复发的高危人群
⑧ 感染（尤其是EBV）是MAS最常见的触发因素
⑨ 即使在生物制剂治疗中也不能忽视MAS风险

4. 停药决策需谨慎
⑩ 临床缓解≠治愈，停药需权衡复发风险
⑪ 停药后应密切随访，一旦复发需尽快重启治疗
