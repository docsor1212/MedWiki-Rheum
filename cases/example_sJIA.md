# 示例病例：系统型JIA (sJIA)

> 本病例为教学示例，展示初诊→检查→诊断→治疗→随访完整流程
> 更新: 2026-04-08

## 基本信息

- 患儿: 张XX，男，5岁8月
- 住院号: JIA-2026-001
- 就诊日期: 2026-01-15

## 主诉

反复高热伴皮疹20天，关节肿痛5天。

## 现病史

患儿20天前无诱因出现高热，体温峰值39.5-40.2℃，每日弛张热（每日≥2次体温>39℃），热退时躯干及四肢出现充血性斑丘疹，热退后皮疹消退。伴食欲下降、精神萎靡。

5天前出现双膝关节肿胀、疼痛，行走困难。发病以来体重下降2kg。否认光过敏、口腔溃疡、雷诺现象。

**既往史**: 体健，预防接种按期完成。
**家族史**: 父母体健，无风湿病家族史。
**过敏史**: 否认。

## 体格检查

- T 39.8℃，HR 120次/分，R 28次/分，BP 95/60mmHg
- 体重 18kg（P25-P50）
- 神志清，精神差，面色苍白
- **皮疹**: 躯干散在橙红色斑丘疹，热退时明显（一过性，Evanescent rash）
- **淋巴结**: 颈部、腋窝可触及数枚肿大淋巴结，最大1.5cm，质中可活动
- **关节**: 双膝关节肿胀、皮温升高、屈伸疼痛，左侧明显；右腕关节轻微肿胀
  - 活动受限关节数: 3（双膝、右腕）
  - 肿胀关节数: 3
  - 压痛关节数: 3
- 肝肋下3cm，脾肋下2cm，无压痛
- 心肺腹查体无特殊

### 影像学

## 初步诊断

### ILAR分类标准对照（sJIA）

| 标准条目 | 符合 |
|----------|------|
| 每日弛张热≥2周 | ✅ 20天 |
| 一过性红斑皮疹 | ✅ 热退时出现 |
| 淋巴结肿大 | ✅ 颈/腋窝 |
| 肝脾肿大 | ✅ |
| 关节炎 | ✅ 3个关节 |
| 排除其他疾病 | ✅ 感染/恶性/其他自身免疫 |

**诊断: 系统型幼年特发性关节炎 (sJIA)**

### MAS风险评估

- HScore评估: 高风险（铁蛋白↑↑↑、血小板↑、FIB↑、肝酶↑）
- 密切监测铁蛋白、血小板、纤维蛋白原变化

## 治疗方案

### 药物治疗

| 药物 | 剂量 | 途径 | 频率 | 疗程 |
|------|------|------|------|------|
| IVIG | 2g/kg | IV | 单次 | 10天内 |
| 泼尼松 | 2mg/kg/d（36mg/d） | PO | qd | 至少4周 |
| 泼尼松减量 | | | | 4周后每2周减10% |
| 叶酸 | 5mg/wk | PO | 每周1次 | MTX给药后3天补充 |

### 病情观察

- q4h监测体温
- 每日监测肝功能、血小板、铁蛋白（MAS筛查）
- 48h评估IVIG疗效

## 随访经过

### 第2周（2026-01-29）

- 体温高峰降至38.0℃，皮疹减轻
- 关节肿痛改善（双膝好转，右腕缓解）
- ESR 52 mm/h，CRP 68 mg/L，铁蛋白 890 ng/mL
- WBC 15.2×10⁹/L，PLT 380×10⁹/L
- **疗效评估: 部分有效**
- 加用MTX 15mg/m²/wk（12.5mg/wk）口服

### 第4周（2026-02-12）

- 体温正常，皮疹消退
- 关节: 双膝轻微肿胀，右腕正常
- 活动关节数: 1（左膝）
- ESR 28 mm/h，CRP 18 mg/L
- **JADAS-27: 12（高活动度）**
- 泼尼松减至1.5mg/kg/d（27mg/d）
- 继续MTX治疗

### 第12周（2026-04-09）

- 无发热，无皮疹
- 关节: 左膝无肿胀，活动正常
- 活动关节数: 0
- ESR 12 mm/h，CRP 4 mg/L
- **JADAS-27: 2（低活动度→接近缓解）**
- 泼尼松减至0.5mg/kg/d（9mg/d）
- MTX耐受良好，无不良反应

### 治疗计划

- 继续MTX 12.5mg/wk
- 泼尼松逐步减量（每月减10%）
- 3月后如达到缓解，考虑MTX单药维持
- 如MTX不能维持缓解，考虑IL-6拮抗剂（托珠单抗）

---

## 病例要点总结

1. sJIA以高热+一过性皮疹为特征，需排除感染和恶性疾病
2. MAS是sJIA最严重并发症，需密切监测铁蛋白/血小板/FIB
3. 治疗策略: IVIG + 糖皮质激素 → MTX → 必要时生物制剂
4. JADAS-27用于客观评估疾病活动度
5. 激素减量需缓慢，联合csDMARD可降低反弹风险
