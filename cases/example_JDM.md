# 示例病例：幼年特发性皮肌炎 (JDM)

> 本病例为教学示例，展示初诊→检查→诊断→治疗→随访完整流程
> 更新: 2026-04-08

## 基本信息

- 患儿: 陈XX，女，6岁5月
- 住院号: JDM-2026-001
- 就诊日期: 2026-02-05

## 主诉

面部及四肢皮疹1月，上下楼梯困难、蹲下后不能站起2周。

## 现病史

患儿1月前无明显诱因出现面部皮疹，以上眼睑紫红色水肿性红斑为著（向阳疹），逐渐延及面颊、颈部及上胸部V形区。日光照射后皮疹加重。双手指关节伸侧（近端指间关节、掌指关节）出现紫红色丘疹，轻度脱屑，偶有瘙痒。

2周前患儿诉双腿酸痛，逐渐加重，表现为上下楼梯困难、从蹲位起立需扶物、平地行走变慢。双手上举困难（梳头、脱衣需家长帮助）。无发热、关节肿痛、吞咽困难、声音嘶哑。

**既往史**: 体健，按期接种疫苗。
**家族史**: 父母体健，无自身免疫病家族史。
**过敏史**: 否认。

## 体格检查

- T 36.8℃，HR 95次/分，R 24次/分，BP 100/65mmHg
- 体重 21kg（P50-P75）
- 神志清，步态缓慢，呈轻度"鸭步"步态，Gowers征阳性

### 皮肤表现

- **向阳疹（Heliotrope rash）**: 双上眼睑紫红色水肿性红斑，边界不清，伴轻度眶周水肿
- **Gottron丘疹**: 双手第2-5掌指关节伸侧及近端指间关节伸侧紫红色扁平丘疹，表面可见细小鳞屑
- **Gottron征**: 双肘关节伸侧、双膝关节伸侧对称性紫红斑片
- **颈部及上胸部V字区**: 毛细血管扩张性红斑
- **甲皱襞微血管改变**: 可见甲皱毛细血管扭曲、扩张（需皮肤镜确认）

### 肌力评估（Medical Research Council, MRC分级）

- **近端肌力**: 3+-4级（对称性近端肌无力）
- **远端肌力**: 4-5级（正常或轻度受累）
- **Gowers征**: 阳性

### 肌电图 (EMG)

- **结果**: 肌源性损害
  - 运动单位电位时限缩短、波幅降低
  - 插入电位延长，可见纤颤电位和正锐波
  - 病变以近端肌（股四头肌、三角肌）为主

### 肌肉MRI

- 双侧大腿近端肌群（股四头肌、内收肌群）弥漫性水肿信号（STIR高信号），对称性分布
- 皮下组织水肿增厚
- 未见明显脂肪浸润或肌肉萎缩（提示急性期，无慢性改变）

### 肌肉活检（左侧股四头肌）

- **组织病理**: 
  - 肌纤维大小不一，散在坏死肌纤维
  - 血管周围及肌束间CD4⁺T淋巴细胞浸润
  - 肌束膜（perimysial）炎症为主
  - 束周萎缩（perifascicular atrophy）特征性改变
  - MHC-I类分子在肌纤维表面弥漫性上调表达
  - 补体膜攻击复合物（MAC/C5b-9）沉积于毛细血管壁
- **病理诊断**: 符合皮肌炎改变

### 恶性肿瘤筛查（抗TIF1-γ阳性，需筛查）

- 腹部B超: 未见异常
- 胸部CT: 未见异常
- 肿瘤标志物（AFP、CEA、CA19-9）: 正常

## Bohan-Peter诊断标准对照（1975）

| 标准条目 | 符合 | 说明 |
|----------|------|------|
| 1. 对称性近端肌无力 | ✅ | 双侧肩带+骨盆带肌力3+-4级 |
| 2. 血清肌酶升高 | ✅ | CK 2850, Aldolase 32 |
| 3. 肌电图示肌源性损害 | ✅ | 近端肌源性损害 |
| 4. 肌活检示炎症性肌病特征 | ✅ | 束周萎缩+血管周围炎症+MAC沉积 |
| 5. 特征性皮肤表现 | ✅ | 向阳疹+Gottron丘疹 |

**满足5/5项标准 → 确诊皮肌炎**

**注**: 满足3-4项+皮肤表现→很可能皮肌炎；满足2项+皮肤表现→可能皮肌炎

## 诊断

1. **幼年特发性皮肌炎 (JDM)** — Bohan-Peter标准 5/5项，确诊
2. 抗TIF1-γ抗体阳性（需长期随访，排除潜在恶性肿瘤）
3. 疾病活动度评估: **Childhood Myositis Assessment Scale (CMAS): 38/52分**（中重度活动）

## 治疗方案

### 诱导缓解期

| 药物 | 剂量 | 途径 | 频率 | 疗程 |
|------|------|------|------|------|
| 甲泼尼龙冲击 | 30mg/kg/d（630mg/d，最大1000mg/d） | IV | qd×3天 | 3天 |
| 泼尼松（冲击后） | 2mg/kg/d（42mg/d） | PO | qd | 4周后减量 |
| 甲氨蝶呤(MTX) | 15mg/m²/wk（皮下注射） | SC | 每周1次 | 长期 |
| 叶酸 | 1mg/d | PO | 非MTX日 | |
| IVIG | 2g/kg（分2天） | IV | 每月1次×3月 | 重症推荐 |
| 羟氯喹 | ≤6.5mg/kg/d（130mg/d） | PO | bid | 皮肤受累 |
| 钙片+维生素D | 钙600mg+D₃ 400IU/d | PO | qd | |
| 防晒 | SPF50+广谱防晒霜 | 外用 | 每日 | 避免日光加重皮疹 |

### 治疗注意事项

- MTX皮下注射优于口服（生物利用度更高、胃肠反应更少）
- IVIG用于重症病例（快速控制肌炎+减少激素用量）
- 皮肤防晒是基础治疗，避免紫外线加重皮疹
- 每4周监测肝功能（MTX肝毒性）和肌酶谱

- 维持缓解状态，活动量恢复正常

- 恶性肿瘤筛查复查: 腹部B超、胸片正常
- 计划MTX继续维持至少1年，缓解≥6月后考虑逐步减量

## 病例要点总结

1. JDM核心三联征: 对称性近端肌无力 + 特征性皮疹（向阳疹/Gottron丘疹）+ 肌酶升高
2. Bohan-Peter标准5条满足≥4条（含皮疹）即可确诊
3. **Gottron丘疹+向阳疹** 是皮肌炎最具特异性的皮肤征象
4. 肌活检: 束周萎缩+血管周围炎症是JDM的病理特征
5. MSA（肌炎特异性抗体）有助于分层: 抗TIF1-γ需筛查恶性肿瘤
6. 治疗方案: IVMP冲击 → 口服激素 + MTX + IVIG（重症）
7. MTX是JDM首选csDMARD，皮下注射优于口服
8. IVIG对重症JDM有效（快速改善肌力和皮疹）
9. CMAS是JDM特异性评估工具，52分满分
10. 长期监测: 肌酶谱变化滞后于临床改善，不能单独作为减药依据
11. 紫外线防护是JDM基础治疗，忽视防晒可导致皮疹反复
