🔴 病例：JDM延迟识别 — 钙质沉着合并双侧髓质肾钙质沉着

儿童患者 — 以广泛钙质沉着首诊，揭示JDM罕见并发症
更新: 2026-04-10

👤 基本信息
患者儿童（具体年龄文献未明确标注，就诊时以广泛钙质沉着为主诉）
就诊机构Sangeetha G等报道，印度
起病表现以广泛钙质沉着（Calcinosis Cutis，皮肤钙质沉着）为主诉就诊
标签延迟识别 钙质沉着 肾钙质沉着 罕见并发症
本病例独特性：这是全球首例报道的JDM合并钙质沉着+双侧髓质肾钙质沉着（Bilateral Medullary Nephrocalcinosis，双侧肾脏髓质钙盐沉积）的病例。肾脏受累在SLE等自身免疫病中常见，但在JDM中极为罕见。本例以广泛钙质沉着首诊，近端肌无力等JDM特征在体检中才被发现，提示JDM的延迟识别是钙质沉着发展的重要因素。
🔴 临床经过 — 延迟识别的教训
首诊 — 以"广泛皮下结节"就诊
患者因广泛皮肤硬结和结节就诊，主要分布于：
• 肘关节伸侧：质硬、不规则的钙化结节
• 膝关节：多处皮下钙化团块
• 臀部和大腿：弥漫性钙质沉着
• 手指：点状钙化

既往被误诊为"皮下囊肿"或"脂肪瘤"。
延迟诊断 广泛钙质沉着
详细体检 — 发现JDM线索
仔细体格检查发现被忽略的JDM特征：
• 近端肌无力：轻微但可检出（上楼稍费力，下蹲站起略困难）
• Gottron丘疹：关节伸侧淡紫色色素沉着（已部分消退）
• 甲褶毛细血管异常：甲周红斑和毛细血管扩张

追问病史：数年前曾有"面部皮疹"和"乏力"的病史，未就诊。
JDM特征被忽略
辅助检查 — 确认JDM+发现肾钙质沉着
实验室检查：
• 肌酶：CK轻度升高
• ANA：阳性
• 炎症指标：ESR/CRP轻度升高

影像学：
• X线：四肢广泛钙化影，皮下组织大量钙质沉着
• 大腿MRI：肌肉水肿信号 + 钙化灶

肌肉活检：肌束间淋巴细胞浸润伴浆细胞，确诊肌炎

腹部超声（常规筛查）：双侧髓质肾钙质沉着 — 意外发现！
双侧肾钙质沉着 JDM确诊
治疗与转归
方案：口服激素 + MTX + 钙质沉着局部处理
肾脏：定期监测肾功能和尿钙，补充枸橼酸钾（Citrate，枸橼酸盐）
结果：肌炎改善，钙质沉着部分消退，肾功能稳定
肌炎改善 · 肾功能稳定
📊 JDM钙质沉着的临床分型
分型特征发生率与JDM的关系
局限性（Localized）小范围皮下钙化结节20-40%最常见，与治疗延迟相关
广泛性（Calcinosis Universalis）弥漫性皮下/筋膜钙化5-10%严重亚型，本例属此型
肿瘤样（Tumoral）大的钙化团块罕见可压迫神经血管
骨外/内脏（Extraskeletal）内脏器官钙化极罕见本例的肾钙质沉着属此类
🔬 JDM钙质沉着的病理机制（深度解析）
🧬 钙质沉着的形成机制
慢性炎症 + 组织损伤 → 坏死细胞释放磷酸盐和钙离子
　↓
局部微环境改变 → pH升高 + 磷酸钙过饱和 → 羟基磷灰石结晶沉积
　↓
巨噬细胞吞噬结晶 → 释放IL-1β（炎症小体激活）→ 更多炎症 → 正反馈
　↓
钙化扩大 → 皮下结节/团块 → 继发感染/溃疡/关节活动受限

关键促进因素：① 持续炎症未控制 ② 延迟诊断 ③ 组织坏死和修复失衡 ④ 抗核浆蛋白抗体（anti-p155/140）
1. 为什么延迟诊断会导致钙质沉着？— 时间依赖的级联反应

第1阶段（0-6月）：IFN-I驱动的肌肉/皮肤炎症
① IFN-I → 内皮损伤 → 血管渗漏 → 肌肉/皮肤炎症
② 如果及时治疗（<6月），炎症可逆，不遗留钙化

第2阶段（6-12月）：慢性炎症 → 组织重塑
③ 持续炎症 → 坏死肌纤维和脂肪组织被巨噬细胞清除
④ 巨噬细胞释放骨桥蛋白（Osteopontin）和BMP-2 — 促进钙化的关键因子
⑤ 成纤维细胞被诱导分化为成骨样细胞（异位骨化的前奏）

第3阶段（>12月）：钙化不可逆
⑥ 磷酸钙结晶在坏死组织中沉积 → 形成羟基磷灰石
⑦ 钙化一旦形成，药物很难逆转，常需手术切除
⑧ 这就是为什么本例患者以"广泛钙质沉着"首诊 — JDM在数年前未被识别

2. 肾钙质沉着为何在JDM中极罕见？— 机制假说
假说机制支持证据
A. 钙磷代谢紊乱广泛钙质沉着 → 软组织钙消耗磷酸盐 → 血磷波动 → 肾脏磷酸盐排泄↑ → 肾小管内钙磷沉积本例腹部超声为常规筛查发现
B. 长期卧床肌无力导致活动减少 → 骨骼失用性脱钙 → 高钙血症 → 肾钙盐沉积JDM患者活动受限常见
C. 药物相关长期激素 → 骨质疏松 → 高钙尿症 → 肾钙质沉着钙质沉着患者常接受长期激素治疗
D. 自身免疫介导IFN-I直接损伤肾小管上皮 → 局部微环境改变 → 钙盐沉积缺乏直接证据
临床启示：JDM患者即使无明显肾损害症状，也应定期行腹部超声筛查。肾钙质沉着可能无症状，但可进展为肾功能不全。延迟诊断是钙质沉着的最重要危险因素 — 早期识别和治疗JDM是预防不可逆钙化的关键。
🆚 钙质沉着 vs 肾钙质沉着 — 机制对比
特征皮肤钙质沉着肾髓质钙质沉着
发生部位皮下/筋膜/肌肉肾髓质（肾锥体）
核心驱动慢性炎症+组织坏死+异位钙化钙磷代谢紊乱+高钙尿症
与JDM活动度的关系正相关（活动度越高越容易形成）间接相关（通过代谢紊乱）
可逆性差（一旦形成很难药物逆转）可能改善（控制病因+枸橼酸盐）
在JDM中的发生率20-40%极罕见（本例为首例报道）
临床后果疼痛、溃疡、感染、活动受限可进展为肾功能不全
📚 临床要点总结
诊断层面：
① 延迟诊断是钙质沉着的最大危险因素 — 早期识别JDM至关重要
② 不明原因的皮下钙化应筛查JDM（尤其儿童）
③ 近端肌无力可能轻微，需要仔细检查

筛查层面：
④ JDM患者应常规腹部超声筛查，即使无泌尿系统症状
⑤ 钙质沉着患者需监测肾功能、尿钙/磷、甲状旁腺激素

治疗层面：
⑥ 钙质沉着的治疗以控制基础疾病活动为首要目标
⑦ 广泛钙质沉着可能需要手术切除或体外冲击波碎石
⑧ 枸橼酸盐可用于预防和治疗肾钙质沉着
⑨ anti-p155/140抗体阳性者钙质沉着风险更高
