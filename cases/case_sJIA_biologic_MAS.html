⚠️ 病例3：生物制剂治疗中发生MAS

2例sJIA患者 — Anakinra/Tocilizumab治疗中EBV感染触发MAS
更新: 2026-04-10
⚠️ 免责声明　本病例基于已发表的医学文献整理，已做脱敏处理。仅供医学专业人员学习参考，不构成任何诊疗建议，不可替代临床医师的专业判断。具体诊疗方案应由执业医师根据患者实际情况制定。如有疑问请咨询专业医疗机构。
📋 病例A：Anakinra（阿那白滞素）治疗后发生MAS
患者女性，9岁诊断sJIA
初始治疗泼尼松 → MTX（甲氨蝶呤）（关节炎持续活动）
标签Anakinra相关MAS 感染触发 环孢素+激素挽救成功
📅 病例A时间线
9岁 — 初诊sJIA
根据ILAR标准诊断为sJIA。泼尼松+MTX治疗，关节炎持续活动。
数年后 — 启动Anakinra
因MTX疗效不佳，启动Anakinra（IL-1受体拮抗剂）。
治疗5周后因疑似化脓性关节炎停药（后续排除感染）。
感染恢复后1月 — Anakinra重新启动
感染问题解决后重新启动Anakinra。
重新启动后第8天 — 肝功能恶化
ALT/AST进行性升高，随后出现MAS典型临床表现：
• 持续高热
• 肝脾肿大
• 铁蛋白显著升高、甘油三酯升高、纤维蛋白原降低
MAS确诊
MAS治疗与转归
方案：环孢素A + 大剂量泼尼松
结果：临床逐步改善，完全缓解。
后续转换至托珠单抗（tocilizumab，抗IL-6受体单抗），临床效果好，无新MAS发作。
关键启示：Anakinra重新启动后8天即出现MAS。推测机制：IL-1通路阻断后，感染→细胞因子网络失衡→替代通路激活→巨噬细胞过度活化。转换为抗IL-6通路（托珠单抗）后MAS未再发，提示换靶点可能是合理策略。
📋 病例B：托珠单抗治疗中EBV感染触发MAS
患者男性，2岁诊断sJIA
特殊病史起病时即以MAS为首发表现（高活动期）
标签Tocilizumab相关MAS EBV感染触发 甲泼尼龙+环孢素挽救
📅 病例B时间线
2岁 — 以MAS起病
根据ILAR标准诊断sJIA，起病时即并发MAS（典型临床表现+实验室+组织学确诊）。
4岁 — 依那西普（etanercept，TNF-α抑制剂）
启动依那西普治疗。1年后因疗效不佳停用。
8岁 — Anakinra有效
启动Anakinra，临床反应良好。维持至14岁。
14岁时因临床缓解+局部不良反应（注射部位疼痛）停药。
停药6个月后 — 疾病复发
新的关节炎发作+系统性活动（发热等）。
启动托珠单抗（抗IL-6受体单抗）单药治疗（未联合糖皮质激素）。
托珠单抗第2次输注后 — 上呼吸道感染 → MAS
第2次托珠单抗输注后出现上呼吸道感染，口服阿莫西林+非甾体抗炎药。
数日后因高热+临床和实验室指标符合MAS入院。
骨髓穿刺确认：噬血现象阳性
微生物检测：近期EB病毒（Epstein-Barr病毒）感染
MAS治疗与转归
方案：环孢素A + 静脉甲泼尼龙冲击
结果：症状完全缓解。
🔍 两例综合讨论
1. 生物制剂治疗中的MAS — 矛盾但非罕见
① IL-1抑制剂（Anakinra）和IL-6抑制剂（托珠单抗）本用于治疗sJIA，但部分患者在治疗中仍发生MAS
② 机制假说：特定细胞因子通路被阻断后，其他促炎通路代偿性过度激活
③ 感染是关键触发因素（本两例均有感染诱因）

2. EBV感染与MAS
④ EB病毒是sJIA-MAS最常见的感染触发因素之一
⑤ 基础免疫异常（sJIA）+ 感染 → 细胞因子风暴 → MAS

3. 治疗策略
⑥ 环孢素A + 糖皮质激素冲击是生物制剂相关MAS的标准挽救方案
⑦ 病例A提示：同一患者从IL-1抑制剂转换为IL-6抑制剂后MAS未再发
⑧ 病例B提示：有MAS病史的患者是MAS复发的高危人群

4. 临床警示
⑨ sJIA患者使用生物制剂期间出现发热，不能简单视为"感染"
⑩ 必须监测铁蛋白、纤维蛋白原、甘油三酯、血细胞计数
⑪ 有MAS病史的患者应格外警惕
