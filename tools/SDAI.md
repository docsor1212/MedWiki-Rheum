# SDAI 简明疾病活动度指数

> ⚠️ 辅助参考，以临床指南和医生判断为准 | 置信度上限: 90%


## 公式
SDAI = TJC + SJC + CrP(mg/dL × 10) + PhGA(0-10) + PtGA(0-10)

- **TJC**: 压痛关节数（28/66/68关节）
- **SJC**: 肿胀关节数
- **CrP**: CRP(mg/dL)，×10后参与计算
- **PhGA**: 医生整体评估 (0-10)
- **PtGA**: 患者整体评估 (0-10)

## 活动度分级
| SDAI | 活动度 |
|------|--------|
| ≤ 3.3 | 缓解 |
| 3.3 - 11.0 | 低 |
| 11.0 - 26.0 | 中 |
| > 26.0 | 高 |

## 优势
- 不需要查ESR
- 计算简单（无需对数）
- 与DAS28高度相关
