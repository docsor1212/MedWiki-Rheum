# CHAQ 儿童健康评估问卷

> ⚠️ 辅助参考，以临床指南和医生判断为准 | 置信度上限: 90%


## 适用年龄
1-18岁，由家长协助填写

## 评估维度（8个功能区）

| 功能区 | 内容 |
|--------|------|
| 穿衣 | 系扣子、拉拉链、穿衣 |
| 起身 | 从椅子上站起、起床 |
| 进食 | 用餐具、端杯子 |
| 行走 | 走路、跑步、爬楼梯 |
| 卫生 | 洗手、洗澡、梳头 |
| 提物 | 提书包、拿重物 |
| 握力 | 开瓶盖、开门 |
| 活动 | 跑步、体育课、骑车 |

## 评分
- 每项0-3分：0=无困难，1=有些困难，2=很多困难，3=做不到
- **CHAQ-DI** = 8项平均分（0-3）
- 0 = 完全正常，3 = 严重功能障碍

## 疼痛和总体评估（VAS 0-100mm）
- 疼痛VAS
- 患者总体VAS
- 家长总体VAS
