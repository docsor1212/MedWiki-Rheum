# 智能随访计划

> ⚠️ 本工具仅辅助参考，最终决策以临床指南和医生判断为准
> 置信度上限: 90% | 所有参数需结合患者实际情况


## JIA 随访方案

### 稳定期（低/不活动）
| 频率  | 检查项目                      |
| --- | ------------------------- |
| 每3月 | 关节检查 + JADAS-27 + ESR/CRP |
| 每3月 | CBC + 肝功（用MTX/生物制剂时）      |
| 每6月 | 身高体重 + 生长发育评估             |
| 每6月 | 眼科检查（ANA阳性时）              |
| 每年  | 疫苗评估 + 骨密度（用激素>6月时）       |

### 活动期
| 频率 | 检查项目 |
|------|----------|
| 每4周 | 关节检查 + JADAS-27 + ESR/CRP |
| 每4周 | CBC + 肝功 |
| 每3月 | 治疗反应评估 + 调整方案 |
| 视需要 | 影像学（关节超声/MRI） |

## JSLE 随访方案

### 稳定期
| 频率 | 检查项目 |
|------|----------|
| 每3月 | SLEDAI-2K + 补体C3/C4 + 抗dsDNA |
| 每3月 | CBC + 肝肾功能 + 尿常规 |
| 每3月 | 尿蛋白/肌酐比值 |
| 每6月 | 生长发育 + 血压 |

### 活动期
| 频率 | 检查项目 |
|------|----------|
| 每2-4周 | SLEDAI-2K + 补体 + 抗dsDNA |
| 每2-4周 | CBC + 肝肾功 + 尿常规 |
| 每3月 | 肾脏评估（必要时肾活检） |
