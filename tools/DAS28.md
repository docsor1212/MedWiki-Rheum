# DAS28 评分

> ⚠️ 辅助参考，以临床指南和医生判断为准 | 置信度上限: 90%


## 公式
DAS28 = 0.56 × √(T28) + 0.28 × √(SW28) + 0.70 × ln(ESR) + 0.014 × GH

- **T28**: 28个关节中压痛关节数
- **SW28**: 28个关节中肿胀关节数
- **ESR**: mm/h
- **GH**: 患者整体健康评估 (0-100mm VAS)

## 28关节列表
双侧: 肩、肘、腕、MCP(1-5)、PIP(1-5)、膝

## 疾病活动度分级
| DAS28 | 活动度 |
|-------|--------|
| ≤ 2.6 | 缓解 (Remission) |
| 2.6 - 3.2 | 低活动度 |
| 3.2 - 5.1 | 中活动度 |
| > 5.1 | 高活动度 |

## 注意
- DAS28主要用于**成人**RA
- 儿童JIA首选JADAS-27
- 但可用于青少年患者（>13岁）参考
