# CDAI 临床疾病活动度指数

> ⚠️ 辅助参考，以临床指南和医生判断为准 | 置信度上限: 90%


## 公式
CDAI = TJC + SJC + PhGA(0-10) + PtGA(0-10)

## 与SDAI区别
CDAI = SDAI - CRP，**不需要任何实验室检查**

## 活动度分级
| CDAI | 活动度 |
|------|--------|
| ≤ 2.8 | 缓解 |
| 2.8 - 10.0 | 低 |
| 10.0 - 22.0 | 中 |
| > 22.0 | 高 |

## 优势
- 最快速的临床评估（床边即可完成）
- 无需抽血
- 适合基层/快速随访
