# JADAS-27 评分

> ⚠️ 本工具仅辅助参考，最终决策以临床指南和医生判断为准
> 置信度上限: 90% | 所有参数需结合患者实际情况


## 公式

JADAS-27 = 活动关节数(0-27) + 医生整体评估(0-10cm VAS) + 患者或家长整体评估(0-10cm VAS) + ESR(mm/h，归一化至0-10)

### ESR归一化
- ESR ≤ 20 mm/h → 0分
- ESR > 20 mm/h → (ESR/10)，上限10分

## 疾病活动度分级

| JADAS-27分值 | 活动度 |
|-------------|--------|
| ≤ 1.0 | 不活动 (Inactive) |
| 1.1 - 3.8 | 低活动度 (Low) |
| 3.9 - 10.5 | 中活动度 (Moderate) |
| > 10.5 | 高活动度 (High) |

## 临床意义
- **目标**: 达到不活动状态或低活动度
- **treat-to-target**: 每3个月评估，调整治疗至达标
