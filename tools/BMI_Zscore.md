# BMI 与 Z 评分

> ⚠️ 辅助参考，以临床指南和医生判断为准 | 置信度上限: 90%


## BMI 计算
BMI = 体重(kg) / 身高(m)²

## WHO儿童生长标准（Z评分解读）

### BMI Z评分
| Z值 | 分类 |
|------|------|
| < -2.0 | 消瘦 |
| -2.0 ~ +1.0 | 正常 |
| +1.0 ~ +2.0 | 超重 |
| ≥ +2.0 | 肥胖 |

### 身高 Z评分
| Z值 | 分类 |
|------|------|
| < -2.0 | 生长迟缓 |
| -2.0 ~ +2.0 | 正常 |
| > +2.0 | 过高（需排查） |

### 体重 Z评分
| Z值 | 分类 |
|------|------|
| < -2.0 | 低体重 |
| -2.0 ~ +2.0 | 正常 |
| > +2.0 | 超重 |

## 儿童风湿免疫常用

### 生长监测要点
- **JIA**长期激素治疗 → 监测身高Z（骨龄+身高）
- **JSLE** → 身高/体重/BMI每6月
- **JDM** → 体重变化反映疾病活动

### 激素对生长的影响
- 泼尼松 >0.1mg/kg/日持续>3月 → 生长减速
- 建议每6月测身高Z，偏差>-1.0需干预
- 考虑生长激素评估（骨龄片）

## 简化Z评分计算
Z = (实测值 - 同龄同性别均值) / 标准差
