# 自身抗体滴度解读

> ⚠️ 辅助参考，以临床指南和医生判断为准 | 置信度上限: 90%


## ANA（抗核抗体）

### 检测方法
- IIF（间接免疫荧光）为金标准
- 滴度报告：1:40, 1:80, 1:160, 1:320, 1:640...

### 临床意义
| 滴度 | 意义 |
|------|------|
| < 1:80 | 通常无临床意义 |
| 1:80-1:160 | 低滴度阳性，需结合临床 |
| 1:320-1:640 | 中高滴度，意义较大 |
| > 1:640 | 高滴度阳性 |

### 荧光模式
| 模式 | 相关疾病 |
|------|----------|
| 均质型 | SLE、药物性狼疮 |
| 斑点型 | SLE、干燥综合征、混合性结缔组织病 |
| 核仁型 | 硬皮病 |
| 着丝点型 | CREST综合征 |
| 环状型 | SS-A/SS-B阳性 |

### JIA中ANA
- 少关节型JIA: ANA阳性率~70-80%
- ANA(+) → 需定期眼科筛查（葡萄膜炎风险↑）
- 多关节型: ~30-40%

## 抗dsDNA抗体
| 项目 | 说明 |
|------|------|
| 特异性 | SLE高度特异性（95%+） |
| 活动性 | 滴度与疾病活动度相关 |
| 阴性预测值 | 高，用于排除SLE |
| 监测频率 | 活动期每1-3月，稳定期每3-6月 |

## 抗Sm抗体
- SLE特异性标志（但敏感性低~10-30%）
- 一旦阳性几乎不转阴
- 不用于监测活动度

## 抗CCP抗体
- 类风湿关节炎高度特异性
- 儿童JIA中意义不如成人
- 可用于鉴别JIA多关节型RF(+)

## 补体 C3/C4
| 情况 | C3 | C4 | 意义 |
|------|----|----|------|
| SLE活动 | ↓ | ↓ | 补体消耗，疾病活动 |
| SLE缓解 | 正常 | 正常 | |
| 低补体荨麻疹 | ↓ | ↓ | |
